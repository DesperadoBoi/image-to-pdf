# ImageToPDF

ImageToPDF - Android-приложение для локального создания одного PDF из нескольких изображений.

## Текущее состояние

Проект находится на раннем этапе разработки. Сейчас реализован базовый сценарий выбора изображений, просмотра миниатюр, изменения порядка страниц перетаскиванием, удаления и ручного поворота страниц, выбора размера страницы PDF, режима размещения «Вписать»/«Заполнить», полей, создания PDF, сохранения через системный Android-диалог, открытия созданного PDF и отправки через системное меню.

## Главная идея

- офлайн Android-приложение для создания PDF из нескольких изображений;
- без рекламы;
- без регистрации;
- без отправки пользовательских документов на сервер;
- обработка выполняется локально;
- простой пользовательский сценарий.

## Запланированные возможности

На уровне MVP планируются:

- выбор нескольких изображений;
- изменение порядка страниц, удаление и поворот;
- настройка параметров PDF;
- локальное создание PDF с отображением прогресса;
- сохранение, открытие и отправка результата через системные механизмы Android.

Подробности зафиксированы в [PRODUCT.md](PRODUCT.md).
