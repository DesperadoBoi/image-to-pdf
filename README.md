# ImageToPDF

ImageToPDF - Android-приложение для локального создания одного PDF из нескольких изображений.

## Текущее состояние

Проект находится на раннем этапе разработки. Домашний экран оформлен как dashboard из восьми инструментов, а «Ещё» открывает единый каталог будущих PDF-возможностей. Реально доступны создание PDF из изображений, системная съёмка первой страницы и отдельный инструмент «Сканировать документ». Smart Scan использует CameraX, ручную perspective-рамку, импорт одной страницы из системной галереи и существующий Editor/PDF flow; автоматическое определение границ и auto-capture пока не реализованы. Изображения выбираются во внутренней MediaStore-галерее с Recent, реальными альбомами и сохранением порядка; Camera, Android Photo Picker и Files доступны как безопасные системные источники. Android 14 partial access показывается отдельно от полного доступа. В редакторе доступны миниатюры, полноэкранное редактирование выбранной страницы, гибкое изменение порядка, удаление и ручной поворот. Перед генерацией progressive `PdfExportSheet` показывает имя, компактный выбор качества и свёрнутые параметры; системный `CreateDocument` открывается после «Конвертировать», progress/cancel сохранены. После успеха автоматически открывается `PdfResultFragment` с одноразовым banner, крупным preview, human-readable metadata и Share/Open; Back возвращает к сохранённым страницам.

## Редактор страниц

- обычное нажатие на миниатюру открывает полноэкранный редактор выбранной страницы;
- горизонтальная лента позволяет переходить между всеми страницами документа;
- основной просмотр поддерживает масштабирование и перемещение изображения;
- поворот влево и вправо доступен через компактные кнопки поверх нижних углов preview;
- прямоугольная обрезка доступна как отдельный режим с reset/cancel/apply;
- ручная коррекция перспективы скрыта из обычного Image-to-PDF сценария и используется отдельным Smart Scan review;
- изменения не перезаписывают исходное изображение и одинаково применяются к миниатюре, просмотру и PDF.

## Главная идея

- офлайн Android-приложение для создания PDF из нескольких изображений;
- без рекламы;
- без регистрации;
- без отправки пользовательских документов на сервер;
- обработка выполняется локально;
- простой пользовательский сценарий.

## Запланированные возможности

На уровне MVP планируются:

- выбор нескольких изображений;
- полноэкранный просмотр страницы с увеличением перед созданием PDF;
- изменение порядка страниц, удаление и поворот;
- создание PDF с выбором качества, размера страницы, ориентации и полей;
- локальное создание PDF с отображением прогресса и возможностью отмены;
- сохранение, открытие и отправка результата через системные механизмы Android.

Подробности зафиксированы в [PRODUCT.md](PRODUCT.md).

## Оптимизация PDF

- Создаваемые PDF автоматически оптимизируются по размеру за счёт ограничения растрового разрешения фактической областью изображения на каждой странице.
- Документы обрабатываются локально, без отправки на сервер и без заметного избыточного разрешения bitmap в итоговом PDF.
