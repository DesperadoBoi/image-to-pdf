# ImageToPDF

ImageToPDF - Android-приложение для локального создания одного PDF из нескольких изображений.

## Текущее состояние

Проект находится на раннем этапе разработки. Сейчас реализован базовый сценарий с домашним экраном выбора изображений или съёмки первой страницы камерой и отдельным редактором страниц. В редакторе доступны просмотр миниатюр, добавление страниц из галереи или системной камеры, изменение порядка страниц удержанием и перетаскиванием, удаление, ручной поворот страниц, создание PDF с детальным прогрессом и отменой, сохранение через системный Android-диалог, открытие созданного PDF и отправка через системное меню. Пользовательские настройки макета временно не показываются: PDF создаётся по умолчанию как A4, с размещением «Вписать» и стандартными полями.

## Главная идея

- офлайн Android-приложение для создания PDF из нескольких изображений;
- без рекламы;
- без регистрации;
- без отправки пользовательских документов на сервер;
- обработка выполняется локально;
- простой пользовательский сценарий.

## Запланированные возможности

На уровне MVP планируются:

- выбор нескольких изображений;
- изменение порядка страниц, удаление и поворот;
- создание PDF с параметрами по умолчанию: A4, «Вписать», стандартные поля;
- локальное создание PDF с отображением прогресса и возможностью отмены;
- сохранение, открытие и отправка результата через системные механизмы Android.

Подробности зафиксированы в [PRODUCT.md](PRODUCT.md).
