Область действия

Эта политика описывает обработку данных в Android-приложении ImageToPDF. Текущая версия приложения предназначена для локального создания PDF из изображений и сканирования документов.

Как обрабатываются данные

  • Изображения и документы обрабатываются локально на устройстве.
  • Приложение не отправляет изображения или созданные PDF на сервер разработчика.
  • Приложение не требует регистрации, не создаёт пользовательские аккаунты и не содержит рекламы или аналитики.
  • Приложение не продаёт данные и не передаёт их рекламодателям.
  • В manifest приложения отсутствует INTERNET permission.

Доступ к камере и изображениям

CAMERA используется только для съёмки страниц в сценарии сканирования документов. Доступ к изображениям используется только после системного выбора пользователя или выдачи соответствующего разрешения. При доступе к MediaStore приложение локально читает сведения об изображениях и их содержимое, чтобы показать встроенную галерею и обработать выбранные страницы. При отказе можно использовать доступные системные способы выбора файлов; без разрешения приложение не получает более широкий доступ.

Сохранение и удаление файлов

Созданный PDF сохраняется в место, выбранное пользователем через системный Android-диалог. После сохранения пользователь управляет этим файлом через выбранное хранилище или файловый менеджер.

Временные снимки, созданные самим приложением, хранятся в app-owned каталоге на время рабочей сессии. Они удаляются при отмене, пересъёмке, удалении страницы или очистке сессии в предусмотренных приложением сценариях. Исходные изображения, выбранные из галереи или стороннего document provider, приложением не удаляются.

Пользователь может удалить локальные данные приложения через настройки Android или удалив приложение. Сохранённые во внешнее выбранное место PDF удаляются самим пользователем.

Открытие и отправка PDF

Открытие и отправка PDF выполняются только по действию пользователя через системные механизмы Android. При отправке пользователь самостоятельно выбирает приложение-получатель; дальнейшая обработка файла регулируется политикой выбранного получателя.

Изменения политики и контакты

Политика должна пересматриваться при изменении разрешений, зависимостей или способов обработки данных. Вопросы о конфиденциальности можно направить на указанный контактный email.
