# Версионирование релизов

## Текущая первая версия

- `versionCode = 1`;
- `versionName = "1.0.0"`.

Эта пара используется для первой тестовой загрузки AAB.

## Правила

- `versionCode` — внутреннее целое число Android. Оно увеличивается при каждом новом AAB,
  предназначенном для загрузки в Google Play.
- Повторно загрузить AAB с уже использованным `versionCode` нельзя, даже если изменились
  код приложения, подпись или `versionName`.
- `versionName` — пользовательская версия, отображаемая в интерфейсах Android и Google Play.
- Версия задаётся явно в `app/build.gradle.kts`; автоматическое вычисление из Git не
  используется.
- Изменение только `versionName` не отменяет необходимость увеличить `versionCode` для
  следующего AAB.

Перед каждой сборкой публикационного кандидата значения проверяются в release-конфигурации
и в собранном артефакте.
